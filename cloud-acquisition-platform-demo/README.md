# 云业务获客平台 · 交互 Demo

基于 PRD 讨论的单页 HTML 演示，涵盖业务闭环、七大业务域与 AI 能力。

## 打开方式

在资源管理器中双击 `index.html`，或在浏览器地址栏打开：

```
file:///d:/vibecoding/cloud-acquisition-platform-demo/index.html
```

## 功能说明

| 模块 | 交互 |
|------|------|
| **驾驶舱** | 切换角色视角（销售/售前/财务/客户/管理层） |
| **业务闭环** | 点击节点或「推进下一步」模拟全流程 |
| **获客域** | AI 评分排序、生成跟进邮件草稿 |
| **客户域** | 选择客户分级（一般/高优/战略），查看 AI 分级解释 |
| **报价域** | AI 生成报价、技产销会审清单 |
| **交易域** | OA 审批推进、合同风险扫描 |
| **计费域** | 分账出账、AI 账单解释（带引用 ID） |
| **增长域** | 账单驱动的续费/优化商机 |
| **AI 架构** | 点击展开各层说明 |
| **Agent 编排** | 运行跨域 Agent 链演示 |
| **统一 Copilot** | 右下角对话，支持快捷问题 |

顶部 **「模拟一笔商机」** 可自动推进业务闭环演示。
